# CoffeeHouse_Webiste_using_NodeJs_and_MongoDB

This is a web application for a Coffee website built using Node.js, Express.js, and MongoDB. The application allows users to browse and order different types of coffee products.

## Features

- User registration and authentication.
- Browse different types of coffee products.
- Add products to the cart and place orders.
- Admin panel to manage products, orders, and users.
- Integration with MongoDB for storing user data, product information, and orders.
- Responsive design for mobile and desktop users.
